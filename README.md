## 为了方便开发,这里使用git来管理版本

当要开发新功能,添加分支, 在分支中完成开发,然后在合并会master 然后删除分支
git checkout -b <分支名>
-b 创建并切换

git branch 检查分支

git merge <分支名> 到当前分支

git branch -d <分支名>
